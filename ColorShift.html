Bookmarklets for shifting the colors within web pages

To install the bookmarklets, just drag the links into the Bookmarks bar:
  • Default Colors Shift (the default target background color is "AAAB8A")
  • Custom Colors Shift (Choose a target background color)
The first time these will be used please allow them to run when prompted what to do.

They are adaptations of these related bookmarklets.

I hope you'll enjoy using them. Despite I'm not a JavaScript expert I do welcome any comments / suggestions.
